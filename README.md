# Investment Dashboard

Análisis de Valor — Valuación DCF con tesis de inversión embebidas.

**URL:** https://kikonavarro.github.io/investment-dashboard

Actualizado automáticamente desde [investment-agents](https://github.com/kikonavarro/investment-agents) via GitHub Actions.

## Cómo funciona

- El dashboard se genera con `tools/web_dashboard.py` 
- Cada push a `investment-agents` con cambios en `data/valuations/` dispara un workflow
- El HTML se pushea automáticamente a este repo
- GitHub Pages sirve el contenido en la URL anterior

---

No es recomendación de inversión. Análisis personal para fines educativos.
